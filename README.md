# Wildlife-Conservation-HCI-App

 Wildlife Conservation Learning App (CSIT226)

 Overview
Group project aligned with **UN SDG 15: Life on Land**.  
Designed an educational app to raise biodiversity awareness among students in Indonesia.

---

 Objectives
- Create an **interactive app** for conservation education.  
- Use gamification (quizzes, streaks, badges).  
- Partner with NGOs for donation transparency.  
- Design lo‑fi and hi‑fi prototypes in Figma.



 Tools & Technologies
- Figma (Lo‑fi & Hi‑fi prototypes)  
- Draw.io (diagrams)  
- Leonardo AI (storyboard visuals)  
- Grammarly, Copilot, ChatGPT (proofreading, idea generation)


 My Contributions
- Designed **Persona 1 (Kevin Ang)** and empathy map.  
- Created **storyboard scenarios** with Leonardo AI prompts.  
- Assisted in **UI/UX design decisions** (simplicity, gamification).  
- Proofread and structured documentation.

 Features
- Interactive learning modules (species, habitats, threats).  
- Quizzes & mini‑games.  
- Gamification (points, badges, streaks).  
- Donation & virtual adoption system.  
- Parent/teacher dashboards.
